Objetivo

Demuestra un gran beneficio a la hora de integrar con el sistema de Vagos, es decir, realiza la integración directa con la base de datos para obtener información precisa y puntual.

Tener en mano el catálogo en todas partes,  realizar pedidos y mostrar los productos desde el dispositivo móvil.

Expandir los procesos de negocios desde la aplicación.

Proporciona a su red de vendedores una solución elegante, simple y eficiente.

Tomar la decisión correcta con independencia de dónde estén los vendedores, proporcionándoles siempre, toda la información necesaria y en tiempo real.

Descarga

La descarga se realiza directamente desde la página oficial del PlayStore de Google.

La URL para acceder al servicio de la aplicación es:

Disponible en Google Play: VagosApp

Accediendo a esta URL se puede instalar directamente desde el dispositivo que esté asociado a la cuenta de google.

Login

  1. Ingrese su usuario: el usuario es el mismo del sistema interno de Vagos, se requiere habilitar una opción en el sistema para acceder vía dispositivo móvil.
  2. Ingrese su contraseña: La contraseña es el mismo generado por el sistema interno de Vagos.

Menu

  1. Inicio: Recupera la pantalla principal
  2. Envía a la pantalla de registro de nuevo cliente
  3. Envía a la pantalla de registro de nuevo pedidos
  4. Envía a la pantalla de lista de pedidos realizados (Pendientes, Facturados y Anulados)
  5. Cerrar sesión: Sale del sistema
  6. Sobre Nosotros: Muestra la info y contacto de los desarrolladores de la aplicación móvil.

Registro de Cliente

  1. CI: Se carga el tipo de documento (CI, RUC, DNI)
  2. Nro. Cédula: Se registra el número de documento del cliente
  3. Nombres y Apellidos: Se registra el nombre y apellido completo del cliente
  4. Dirección: Se registra la dirección del cliente
  5. Teléfono: Se registra el número de teléfono de contacto del cliente
  6. Ciudad, Departamento y Barrio: Se registran de la lista disponible.

Registrar Pedido

  1. CI: Se selecciona el tipo de documento
  2. Se pulsa el Icono BUSCAR (Lupa) y el sistema busca la información del cliente en el servidor. Si encuentra, se completa los datos en: Cod.Cliente, Nombres y Apellidos, Dirección, Ciudad y Departamento.
  3. Una vez seleccionado el cliente, aparece el siguiente mensaje:
  5. Se pulsa aceptar para seleccionar el cliente
  7. PASO 1 DE 3: El cliente está seleccionado y la aplicación permite pulsar SIGUIENTE > Para continuar con el proceso de registro de Pedido.
  9. PASO 2 DE 3: Se busca los artículos para cargar al pedido o carrito de compras.
  11. Se carga el código del artículo en el buscador y el sistema obtiene en tiempo real el articulo, permitiendo así carga la cantidad de artículo en el carrito de compras seleccionando con el dedo el artículo resultante del buscador.
  13. La fecha DOWN (Abajo) permite restar la cantidad. La flecha UP (Arriba) Permite aumentar la cantidad de artículo.
  15. Para eliminar o editar un detalle del pedido, se desliza el detalle hacia la izquierda
  17. Para registrar el pedido se pulsa la opcion: PROCESAR ORDEN
  20. Una vez registrado, aparece un mensaje de Proceso Exitoso.
  22. Muestra La lista de los pedidos (Pendientes, Facturados y Anulados)

Conclusión

La evidencia que se mostró anteriormente demuestra que fácilmente se puede realizar pedidos on-line via app integrado a la aplicación interna de Vagos. Esto hace eficiente el control, aprobación y facturación de pedidos en tiempo real.

La flexibilidad de esto permite la toma de decisiones acertadas en forma continua e integrada a todos los procesos de negocios interno de la empresa. La clave principal es la distribución de datos en forma simultánea. Otro punto principal, la escalabilidad horizontal del diseño de los servicios de toma de información a través de la plataforma instalada en los centros de cómputos de Vagos.
